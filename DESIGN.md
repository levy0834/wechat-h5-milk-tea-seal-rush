# DESIGN.md

## Game Name

奶茶封口王

## One-line Sell

看准杯型和时机，在爆单传送带上疯狂封口，手一抖就翻车。

## Inspiration

- 微信 H5 高频短局的分数挑战节奏
- 快反应 + 读图识别类街机感
- 排队爆单、手忙脚乱的奶茶店场景

## Innovation

区别于纯点击或纯躲避，本作要求玩家在极短时间里同时完成：

1. 识别杯型对应盖子颜色（认知任务）
2. 在封口区内点击（时机任务）

形成“识别 + 时机”双任务闭环。

## Core Loop

1. 杯子随机杯型进入传送带
2. 杯子靠近封口区
3. 玩家选择对应盖子并卡点点击
4. 成功得分叠连击；失败翻车扣分并断连
5. 难度随时间抬升，30 秒结算称号与分享文案

## Rules

- 局时：30 秒
- 成功：杯子在封口区内且盖子类型匹配
- 失败：封错盖、空拍、漏封
- 反馈：分数、连击、老板吐槽、震屏、粒子效果
- 结算：本局分、历史最佳、最高连击、段位称号

## Shareability

- 结果页自动生成“吹牛文案”并支持一键复制
- 分数 + 连击 + 称号便于群聊比较
- 单局短，可连续重开冲榜

## UI Style

- 移动端竖屏优先
- 奶茶暖色系背景 + 传送带纹理
- 大按钮、明确色彩语义、强状态反馈
- 卡通轻夸张动画，保持可读性与点击准确性

## MVP Scope

- 单页面静态 H5（`index.html` + `styles.css` + `script.js`）
- 三种盖型与对应按钮
- 30 秒完整流程（开始、游戏中、结算、重开）
- 难度曲线、连击、最佳分本地存储
- 分享文案复制功能
