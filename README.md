# 奶茶封口王

一个微信风格、30 秒一局的 H5 反应小游戏：

> 看准杯型和时机，在爆单传送带上疯狂封口，手一抖就翻车。

## 游戏卖点

- 3 秒上手：杯子进入封口区时，点击对应颜色的封口盖。
- 双任务挑战：不仅要卡时机，还要识别杯型对应的盖子。
- 强回合反馈：连击、翻车、老板嘴炮、称号结算、可复制吹牛文案。
- 适合社交传播：短局制冲榜，分数和称号可以直接分享。

## 玩法说明

- 杯子会沿传送带向上移动，并带有颜色标记：
  - 蓝色 = 冰饮盖
  - 粉色 = 奶盖盖
  - 黄色 = 热饮盖
- 当杯子处于“封口区”时，点击底部对应颜色按钮即可成功封口得分。
- 误点、空点或漏封会翻车扣分并打断连击。
- 单局 30 秒，目标是打出更高分和更高连击。

## 本地运行

无需安装依赖，无构建步骤。

1. 直接双击打开 `index.html`。
2. 或在仓库目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## GitHub Pages 部署说明

仓库已包含 `.github/workflows/deploy-pages.yml`。

- 推送到默认分支后，GitHub Actions 会自动构建并发布静态页面。
- 在仓库设置里启用 Pages（Source 选择 `GitHub Actions`）即可。
